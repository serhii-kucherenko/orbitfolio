# Orbitfolio benchmarks — award interface reboot

Design lab pass — **100** variants scored per `docs/RUBRIC.md`, compared with award-winning portfolio craft (Awwwards, Favourite Website Awards, CSS Design Awards, Codrops — 2025–2026 patterns).

**Hypothesis:** Many structurally different designs + scored promotion + iterative learning beats a single-shot portfolio.

## External craft signals (steal the *pattern*, not the brand)

| Pattern | What awards reward | Orbitfolio design families |
|---------|--------------------|----------------------------|
| Scroll as narrative engine | Camera / scene advances with scroll; one continuous take | Web graphics rooms, warp scrub, alcove museums |
| Restraint + motion craft | Frame-perfect animation; typography weighted; no noisy chrome | Editorial, kinetic type, Swiss, By-Kin-like calm |
| Browser 3D for atmosphere | 3D frames the work; never buries hire proof | Planet stage, shader veil, spotlight installs |
| Spatial browsing | Pan/zoom, rooms, corridors, alcoves | Infinite canvas, isometric rooms, walkable rail |
| Interactive systems | Command palette, micro-games, curiosity gestures | Command palette, agent console, proof mosaic |
| Hire clarity | Who / what / proof in under 10 seconds | Recruiter, clinic product, dual narrative |

## Promotion

Champion (`/`) = highest overall score. Ties → Coolness → Hireability.

## Learning rule

Every Hybrid design documents what it stole from earlier high scorers in `docs/FINDINGS.md`. The next Hybrid must beat the previous one on at least two axes or overall score.
